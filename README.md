# Instacart-Delivery
Proyecto 4- Instacart Delivery, es un proyecto en el que se realizó la limpieza de los diccionarios; lo que permitió realizar un análisis para encontrar a los productos mas populares. Igualmente se visualizaron las distribuciones de consumo de productos por periodos de tiempo . Todo esto para entender el consumo y mejorar estrategias de Marketing.
